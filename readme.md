Descripción del proyecto

Este proyecto consiste en la creación de un sitio web desarrollado con HTML y CSS que incluye un simulador de créditos, una página de productos y una página de solicitud de crédito. El sistema permite navegar entre las diferentes secciones mediante enlaces con equetas a que redirecciona a cada una de las paginas y presenta un diseño amigable y funcional para el usuario

El simulador incluye filtros funcionales los cuales fueron logrados manejando los estilos de cada elemento de la estructura HTML mediante el CSS
